# Project 2: SIMD Matrix Multiplication

#### Building the Project
```
$ cd project2
$ mkdir build
$ cd build
$ cmake ..
$ make
```

#### Running the Project
```
$ ./ACS_Proj2 <square_matrix_size> <num_iterations>
```
